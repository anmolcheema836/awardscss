# AwardsCSS
A premium, pure CSS architectural framework for Awwwards-style, high-end websites. Built with native CSS view-timelines, grid mathematics, and GPU-accelerated masking. No JavaScript timeline libraries required.

## Features
- 🚀 Pure CSS (Zero JS required for animations)
- 🎥 Cinematic Media Reveals (Curtains, Splits, 3D Fan Stacks)
- 📱 Fluid Typography (`clamp()` driven)
- 🖱️ Hardware-accelerated Hover FX & Glassmorphism

You can include the compiled CSS directly into your project:
