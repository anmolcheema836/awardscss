The Complete API.

Every utility inside the AwardsCSS infrastructure follows strict globally rendering naming conventions under the aw- namespace.

1. Installation & Setup

AwardsCSS is officially available on the npm registry and via globally distributed CDNs. Choose your preferred installation method below.

Package Managers

Via NPM

npm install awardscss

Via Yarn

yarn add awardscss

Via PNPM

pnpm add awardscss

If you are using a modern build tool (Vite, Next.js, Vue, etc.), import the compiled CSS directly into your main entry file:

// App.jsx or main.js import 'awardscss/dist/awardscss.css';

Direct CDN Setup (Recommended for Vanilla)

You can instantly leverage the full architecture without any build tools by dropping this stylesheet link directly into your <head> tag.

<!-- jsDelivr Global CDN --> <link href="https://cdn.jsdelivr.net/npm/awardscss/dist/awardscss.css" rel="stylesheet">

2. Base Architecture

To initialize the framework's variables (like fonts, colors, and base easing curves), you must apply the correct classes to your <body> element.

ClassDescription
.theme-darkInitializes the dark mode color palette globally.
.theme-lightInitializes the light mode color palette globally.
.aw-noise-animatedApplies the animated base64 SVG TV static/noise overlay.
<!-- Ensure overflow-x is clip to prevent native horizontal scrollbar bugs --> <body class="theme-dark aw-noise-animated" style="overflow-x: clip;"> <main class="container pt-section"> <!-- Content Goes Here --> </main> </body>

3. Typography & Text FX

We do not use fixed pixels. All typography utilizes CSS clamp() to perfectly scale between mobile and 4K displays.

ClassDescription
.text-xs to .text-displayFluid clamping font scales. No media queries needed.
.font-displayApplies the primary headings font (Clash Display/Oswald).
.font-monoApplies the secondary technical font (Space Mono).
.aw-glitchCyberpunk RGB split. Use data-text="string".
.aw-liquid-fillOutline text that fills vertically on hover like water.
.aw-rollerSlot-machine vertical text roller wrapper on hover.
.aw-inline-reveal-wrapExpands an inline image between text lines.
<!-- Liquid/Wave Text Fill (Needs data-text attribute matching inner text) --> <h1 class="aw-liquid-fill text-display font-display" data-text="Fluid">Fluid</h1> <!-- Slot Machine Roller Text --> <span class="aw-roller"> <span class="aw-roller-inner"> <span class="aw-roller-text">Designers</span> <span class="aw-roller-text text-gradient-accent">Developers</span> </span> </span> <!-- Inline Image Expand --> <span class="aw-inline-reveal-wrap"> Expand your <img src="..." class="aw-inline-reveal-img"> vision. </span>
